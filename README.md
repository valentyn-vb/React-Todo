# React-Todo

Small learning-oriented Todo app built with `Next.js`, `TypeScript`, and `Material UI`.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run typecheck`

## Deployment

1. Push the repository to GitHub.
2. Import the repository into Vercel.
3. Keep the framework preset as `Next.js`.
4. Set the production branch to `main`.
5. Deploy and verify that todos persist after a refresh.
